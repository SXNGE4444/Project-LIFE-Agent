# Installing LIFE Agent on Any Browser

## 🦁 Brave Browser
1. Go to `brave://extensions`
2. Turn ON **Developer mode**
3. Click **Load unpacked**
4. Select the `browser-extension` folder
5. Click the puzzle icon 🧩 → pin LIFE Agent

## 🌐 Google Chrome
1. Go to `chrome://extensions`
2. Turn ON **Developer mode**
3. Click **Load unpacked**
4. Select the `browser-extension` folder
5. Pin the extension

## 🔥 Mozilla Firefox
1. Go to `about:debugging#/runtime/this-firefox`
2. Click **Load Temporary Add-on**
3. Select `build/firefox/manifest.json`
4. The icon appears in your toolbar

## 🟣 Microsoft Edge
1. Go to `edge://extensions`
2. Turn ON **Developer mode**
3. Click **Load unpacked**
4. Select the `browser-extension` folder

## 🔴 Opera
1. Go to `opera://extensions`
2. Turn ON **Developer mode**
3. Click **Load unpacked**
4. Select the `browser-extension` folder

## 🧪 Test It Works
Visit any of these sites to see dark pattern detection:
- amazon.com
- facebook.com
- booking.com
- any e-commerce site

Look for:
- Red dashed borders around dark patterns
- Red warning banner at top of page
- Click the LIFE Agent icon for full report
